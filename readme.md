#### Introduction

    Some testing about CSS layout

### Prerequisite

- node > v12.0.0
- npm > v6.9.0

#### Steps

- npm install

```
npm i
```

- start listening

```
# or yarn dev
npm run dev
```

- view panel

```
curl http://127.0.0.1:3000/index.html
```
